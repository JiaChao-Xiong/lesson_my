- 防抖
    - 因为函数需要防抖优化，不可以直接去执行原本功能的函数
    - 使用debounce通用函数去包装一下原函数
        debounce(ajax, 500);
    - 依赖的是闭包能力
        - 本要执行的函数，闭包到一个函数里 debounceAjax
        - 上一次的定时器id 交给闭包空间
